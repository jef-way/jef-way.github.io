Wox - 开源免费强大的快捷启动器辅助工具

2015-11-21
软件
阅读数量:次

在日常的电脑操作中，我们常常花很多时间重复做着一件事情：退回桌面/开始菜单->寻找软件图标->点击启动。但如果你喜欢高效的键盘操作，那么你需要一款好用的 快速启动器软件。

我们曾介绍过 Launchy、AltRun、Executor、Alfred 等键盘快速启动工具，而今天推荐的则是另外一款国产开源免费的软件快捷启动工具 Wox！它可以搜索你电脑上的程序，通过快捷键呼出搜索框，输入少量字母快速匹配出对应软件并启动、打开网页、翻译单词、或配合 Everything 插件搜索任意文件等等……

Wox - 强大高效的快捷启动软件

你可以将 Wox 看作一个高效的本地快速搜索框，通过快捷键呼出，然后输入关键字来搜索程序进行快速启动，或者搜索本地硬盘的文件，打开百度、Google 进行搜索，甚至是通过一些插件的功能实现单词翻译、关闭屏幕、查询剪贴板历史、查询编程文档、查询天气等更多功能。

相比 Launchy 等同类的快速启动器工具，Wox 最大的特点是可以支持中文拼音的模糊匹配。譬如说你想搜索“设备管理器”，那么只需输入"sheb"就能匹配得到。

增加程序的搜索目录：

一开始使用 Wox 时，你会发现，除了开始菜单里面有的程序能搜索到之外，自己的一些绿色软件或者安装到非默认目录的软件都找不到。这时你可以在 Wox 里输入 settings 回车或托盘图标右键选择 Settings 进入设置界面，然后在 Programs 中按 Add 按钮添加你的软件所在的文件夹。

如下图，Type 一般选择 Portable 开头的那项，然后在 Location 里输入目录路径，BounsPoint 是优先级，默认填0即可。设置后退出并重启 Wox 即可生效。

网页搜索功能：

Wox 的网页搜索功能是可以让用户自己为某个搜索引擎设置一个关键字。举个简单例子，如果用户经常使用百度搜索。那么用户就可以在 Wox 的设置里的 Web Search 选项中，点击 Add 增加这么一个选项（URL为http://www.baidu.com/s?wd={q}）：

这样，你便可以通过在 Wox 搜索框里输入「bd 关键字」进行百度搜索。同理，你也可以增一个淘宝搜索、必应搜索，或者任何其他你所需要打开的网站，网址中的{q}是代表你输入的关键字，发挥你的想象力吧。

插件管理：

同样是在 Wox 里键入 settings 并回车进入设置界面，Plugins 一页中便是你目前安装的插件。你可以在这里启用/禁用或者设置你的插件。你可以在这里找到 Wox 的插件下载，下载回来的是 .wox 格式的文件，直接拖放到 Wox 的搜索框上，软件就会提示你是否安装了。

Everything 搜索插件：

Everything 无疑是目前最强大的文件秒搜工具了，因此在 Wox 上集成 Everything 的功能一定会让它变得更酷更实用！但是，直接安装完 Everything 的插件你会发现通过它的命令「ev 文件名」并不能搜索。这时，你还需要进入插件的目录中，将 x64 或 x86 文件夹里面的 Everything.dll 复制一个到上层目录，重启 Wox 才能生效。

更多插件：

目前 Wox 已经有不少实用的插件了，如豆瓣电影、雅虎天气、进程关闭器、URL编码、关闭显示器、剪贴板历史、Windows最近使用文档等等……更多的 Wox 插件下载以及它们使用方式，大家可以前往这里获取，限于篇幅这里就不介绍更多了，大家亲身去尝试吧。

另外，官方的介绍上表示可以通过输入「wpm install 插件名」命令进行插件安装，但我这边实测时不知为何并无此命令的存在，可能在新版本会修正这个问题吧。

其他功能：

运行命令行/替换Win+R：

Wox 还提供了 Win+R 替换运行窗口的功能（可在设置中关闭），也提供了快速输入命令行的功能。也就说在打开Wox的情况下，如果用户按下 Win+R 将会直接打开Wox并进入命令行状态（用户也可以通过输入>直接进入此状态），如下图，键「 >ping www.jefway.com」可以快速执行 ping 命令。

快速计算器：

Wox 还能进行一些简单的数学运算，可以非常方便地随手进行计算，并且回车后还可以复制计算结果。

自定义主题外观：

除了插件外，Wox 还支持自定义主题外观。你可以在设置界面中的 Theme 页面中选择已有的主题，也可以通过这个在线的主题制作工具自行定制。

总结 Wox 的一些特性：

  • 可以搜索本机安装的程序，支持中文拼音模糊搜索
  • 搜索浏览器书签
  • 瞬间搜索出本地硬盘所有文件（通过 Everything 插件）
  • 网页搜索功能，可以给某个网址设置关键字例如搜索google，可以用g 搜索内容
  • 支持插件扩展
  • 支持自定义主题

写在后面：

虽然 Wox 推出的时间并不长，目前还不算很稳定(偶尔出错)，但由于其开源且支持插件扩展，整体思路优秀，个人认为它的潜力非常巨大。由于它借鉴了 Mac 上的效率神器 Alfred 高效的工作方式，有点 Alfred Windows 版 的味道。假以时日，一旦插件丰富起来、稳定性提高之后，Wox 也将会成为 Windows 上的最佳效率神器。
