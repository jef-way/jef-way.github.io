WGestures - 实用的全局鼠标手势工具软件

2016-04-26
软件
阅读数量:次

说到「鼠标手势」大家应该都不会陌生，现在各种先进的浏览器上都自带了或者有相关功能的插件可以安装。它们能让你利用鼠标在屏幕上划出指定的线条来执行对应的动作，操作起来非常方便高效！

当然，这么实用的功能怎能只让浏览器独享呢？今天就给大家介绍一款能为 Windows 整个系统都增加鼠标手势自定义的功能！它们都已经不能很好支持最新的 Windows 10 了。而今天推荐的 WGestures 则是一款全新且免费开源的系统全局鼠标手势软件神器……

WGestures 超级实用的免费全局鼠标手势软件新秀

WGestures 是一款免费且开源的国产软件，作为一个全局的鼠标手势软件，WGestures 可以作用于几乎所有的 Windows 程序！你可以设置手势来执行前进、后退、关闭窗口、最大化、最小化、复制、粘贴、执行命令代码、输出文本、打开网址、运行程序、执行快捷键等等实用的操作。

任何时候，只要你按住鼠标右键划出对应的手势，就能快速执行这些预设的功能。譬如，软件默认情况下，向上划线执行复制，向下粘贴，左下最小化，右上最大化，左下开始菜单，右下显示桌面等等，习惯了之后使用起来真心超级方便高效！

相信当你再根据自己的使用情况加以设置调整之后，WGestures 足以让你大呼用得过瘾！

除了软件预设的一些全局手势之外，你也能添加自定义的手势，这些手势可以全局的，也可以是只针对某款软件。譬如你可以给 Google Chrome 浏览器增加强大的鼠标手势功能，而不必再安装它那些蹩脚又不好用的鼠标手势插件了。如下图按加号，然后按提示增加指定的程序：

当你想要利用 WGestures 的鼠标手势来执行软件的某个功能（如 Chrome 恢复刚关闭的网页），实现的办法就是先找到它的键盘快捷键，这里是 CTRL+SHIFT+T，然后你新增一个鼠标手势来执行这个快捷键即可。这样，无论是上一个标签页(CTRL+TAB)、下一个标签页(CTRL+SHIFT+TAB)、关闭标签页(CTRL+W)、新建隐身窗口(CTRL+SHIFT+N) 什么的各类功能都能轻松通过鼠标手势来完成了。完成设置之后，基本上浏览网站时都已经不需动键盘了，既惬意，效率又特别高！生活多么美好啊……

WGestures屏幕触发角功能

用过苹果 Mac 电脑的同学可能会知道，OS X 上有一个小功能，可以设置屏幕4个触发角的动作，譬如当鼠标指针移动到最右下角时可以触发「最小化」，移动到右上角的话触发「机器睡眠」等各种动作。WGestures 现在也为我们提供了这样的「触发角」功能，您只需将鼠标移到屏幕角落即可触发相应操作。而且您还可以自定义触发角的功能！

写在后面

WGestures 的手势识别率高，稳定性也不错，支持 Windows 10、Win7/8 (兼容32与64位系统)，而且软件安装后自带了“快速入门”使用教程，即使新手也能很快上手。

有了 WGestures，你只需一只手操作鼠标，就能获得极高的操作效率，甚至有时比起记住各种快捷键的键盘操作还要方便！

总的来说，WGestures 是追求高效工作的朋友必备的神器，你不应该错过它！但需要注意的是，如果你经常玩游戏的话，最好是先暂停手势功能，不然偶尔匹配出来的手势可能会对游戏有影响。
