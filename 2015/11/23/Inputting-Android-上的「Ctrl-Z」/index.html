Inputting+ - Android 上的「Ctrl+Z」

2015-11-23
APP
阅读数量:次

和心上人聊天的时候，我们总是斟酌用词、语气严谨，生怕说错什么让对方误解。所以在说话时，我们常常会经历「输入、删除、输入、删除」的往复过程，可是翻来覆去还是觉得第一次说的比较靠谱，于是只好再重新输入一遍。然而当我们发送完这句话，这时候往往已经过去了好几分钟，对方是不是以为我对 TA 爱答不理呢？如果手机上能够做到桌面系统上的「撤销」功能那不就方便多了。

少数派曾经为大家介绍过 Type Machine，它能够记录所有你输入过的文字，然而它也仅仅是记录一段输入完整的文字，不能记录输入的历史动作，想要用它实现「撤销」和「重做」确实有些捉襟见肘。而 Inputting+ 这款应用不仅能够实现 Type Machine 的所有功能，还高精度记录了我们所有的录入动作，从此，我们打字也可以吃后悔药了。

记录一字一符

Inputting+ 的主要目的就是做到 PC 或 MAC 上的「撤销」和「重做」功能，这种功能的原理无非就是系统或应用在后台记录你输入过的东西，然后再需要的时候一次一次恢复回来。

Inputting+ 的运作方式非常简单，当我们点击输入框，Inputting+ 的悬浮球会与输入键盘同时弹出，我们将候选词上屏的时候，Inputting+ 会在后台记录这次上屏内容，下一次再有内容上屏的时候，如果同一个输入框中有内容，那么 Inputting+ 会认为第二次上屏内容为增量输入，从而将这次输入的文字作为撤销的一个步骤。

当我们需要执行撤销或重做动作，只需要点击悬浮球，在弹出框中进行相应的动作，完成后点击返回键，Inputting+ 会通过剪切板自动复制粘贴，快速省事儿。

除了撤销和重做，Inputting+ 还可以进行文字查找，逐个删除或者替换文字的时候非常有用。

放心一步一路

Inputting+ 同样可以实现 Type Machine 的输入时间线功能。Inputting+ 中的输入时间线功能需要付费解锁，支持 Google Play 付款或支付宝支付。支付宝支付结束后点击我已购买即可解锁。开发不易，请大家支持正版。

解锁输入时间线功能后，再次进入 Inputting+ 会直接看到时间线，我们曾经输入过的内容以卡片形式按照时间顺序排列，并且每一个卡片都单独记录了上屏内容，可以直接在卡片中进行撤销或重做，重复输入内容方便至极。

值得一提的是，为了保护隐私，Inputting+ 在输入时间线界面是不允许截图的。无论是使用系统快捷键，或是豌豆荚之类的手机助手，亦或是通过 AirMirro，我们都无法直接获取屏幕快照。Inputting+ 的这点设计值得称赞，可见开发者的用心良苦。

除此之外，Inputting+ 可以与「Clip Stack 剪纸堆」联动，这是一款剪切板历史记录应用，两者相互配合，对于某些重复性较高的文字录入工作非常有用。

撤销和重做功能虽然不起眼，在桌面系统里实现起来也是易如反掌，而在手机端却是我们非常需要的功能。Inputting+ 的出现让我们可以放心的去「后悔」。如果你是一个文字工作者，或者是一位喜欢用手机码字的人，那么 Inputting+ 一定是你需要的工具。
